# NEAR Voting Frontend

A React frontend for the NEAR voting smart contract.

## Features

- Connect NEAR wallet
- Create voting proposals (0.1 NEAR fee)
- View all proposals and results
- Cast votes on active proposals
- Real-time vote counting

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open http://localhost:3000

## Usage

1. Connect your NEAR testnet wallet
2. Create proposals with title, description, options, and deadline
3. Vote on active proposals
4. View results in real-time

The app connects to the deployed contract at `hightdevtest.testnet` on NEAR testnet.
